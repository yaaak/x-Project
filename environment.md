##macOS
1. maven的安装
   - brew install maven
2. man -v (JDK >=7)
3. 利用Maven创建Sprintboot的工程
   - mvn -B archetype:generate -DgroupId=edu.self -DartifactId=my_first_spring_boot -Dversion=1.0.0-SNAPSHOT -DarchetypeArtifactId=maven-archetype-quickstart
4. 运行Sprintboot的工程
   - mvn spring-boot:run
